mkraspbian-chroot
=================

Script for setting-up a Raspbian chroot directory for developing software for the Raspberry Pi (http://raspberrypi.org) on your Linux desktop computer.  The script can also help you bind-mount /dev, /proc and /sys filesystems to enable device access and networking inside your chroot, unmount bind-mounts and start a chroot session or execute a chrooted command.
